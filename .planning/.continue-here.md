## UAT 步驟（下次 session 要跑的）

### Step 1 — 部署 Google Apps Script
1. 複製 `google-apps-script.js` 全文
2. https://script.google.com → 貼進編輯器 → 儲存 → 部署新版本
3. 複製 Web App URL
4. 在編輯器執行 `initSheets` 一次（允許授權）
5. 確認 Google Sheet 出現兩張表：
   - 記錄（15 欄）
   - 工單（6 欄）

### Step 2 — 前端設定
1. 開網站（Vercel 或 GH Pages）→ 管理頁
2. 貼 Web App URL → 儲存
3. 貼工單 Sheet URL → 按「🔄 重新拉工單」

### Step 3 — 煙霧測試
- **測 A**：均質站勾 3 層送出 → 應跳確認視窗 → Sheet 應新增 3 列；單層送出 **不應** 跳視窗
- **測 B**：貼錯 URL 應跳「URL 格式錯誤」；未公開 Sheet 應跳「請設為知道連結的任何人可檢視」
- **測 C**：「🔄 重新拉工單」無變更應跳小提示；有變更應跳差異確認視窗
- **測 D**：分析頁應從雲端讀回剛送出的資料
